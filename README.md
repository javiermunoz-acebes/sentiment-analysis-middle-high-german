# sentiment-analysis-middle-high-german
Análisis híbrido de sentimientos en textos literarios del Alto Alemán Medio usando el léxico SentiMHD. Combina enfoques léxicos, machine learning y reglas para detectar emociones en literatura medieval alemana (Parzival, Tristan, Erec). Notebooks Jupyter reproducibles con corpus anotado y métricas.
